# NodeFlix
 ### Service media interface based on user's files

##  Features
* Posters and infos about the medias
* Genre classification
* Season selector

## Requirements
* Node.Js 12 or latest
## Install
```bash
  npm install
```

## Comand
- Start
```bash
npm start
```
- Development
```bash
npm run dev
```
- Production
```bash
npm run build
```
